Steps to integrate alice ios-skin via zip file generated from build script into the BasicPlaybackSampleApp to create an AliceSkinSampleApp


1. Unzip the zip file generated in `/path/to/ios-skin`

2. Open BasicPlaybackSampleApp in xcode

3. a) Under linked frameworks and libraries in BasicPlaybackSampleApp, add OoyalaSkinSDK.framework from the zip file that you unzipped
   b) Under linked frameworks and libraries, search for and add JavascriptCore.framework and libz.dylib

4. Inside Build Settings under the linking section, add `-ObjC` to the settings that say "Other Linker Flags". Add this flag to both Debug and Release.

5. Drag the main.jsbundle file from the zip file into the xcode project into BasicPlaybackSampleApp > Players

6. a) In BasicPlaybackSampleApp > Players > BasicSimplePlaybackViewController.m, change `#import <OoyalaSDK/OOOoyalaPlayerViewController.h>` to `#import <OoyalaSkinSDK/OOSkinViewController.h>`
   b) Add `#import <OoyalaSDK/OODiscoveryOptions.h> to the list of imports`
   c) In the same file, replace all instances of `OOOoyalaPlayerViewController` with `OOSkinViewController`
   d) replace the line `self.ooyalaPlayerViewController = [[OOOoyalaPlayerViewController alloc] initWithPlayer:player];` (around line 49) with
```
OODiscoveryOptions *discoveryOptions = [[OODiscoveryOptions alloc] initWithType:OODiscoveryTypePopular limit:10 timeout:60];
NSURL *jsCodeLocation = [[NSBundle mainBundle] URLForResource:@"main" withExtension:@"jsbundle"];
self.ooyalaPlayerViewController = [[OOSkinViewController alloc] initWithPlayer:player  parent:self.playerView discoveryOptions:discoveryOptions launchOptions:nil jsCodeLocation:jsCodeLocation];
```

7. a) Drag the fonts folder inside of the zip file into the root of the BasicPlaybackSampleApp xcode project.
   b) Open the info.plist in the root of the BasicPlaybackSampleApp and add a field with the key 'fonts provided by application' and array elements of the filenames inside the fonts folder (eg alice.ttf)
